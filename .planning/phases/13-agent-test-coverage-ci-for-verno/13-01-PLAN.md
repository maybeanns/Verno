# 13-01: MockLLMService + Agent Unit Tests

## Goal
Implement a deterministically injectable `MockLLMService` and deploy test coverage for the core agent orchestrations in `src/test/suite`.

## Tasks

### 1. Implement MockLLMService

<read_first>
- src/services/llm/providers/AnthropicProvider.ts
- src/types/index.ts
</read_first>

<action>
Create `src/services/llm/providers/MockLLMService.ts` that implements `ILLMProvider`.
It must contain:
1. `private responses: string[] = []`
2. `public enqueueResponse(res: string) { this.responses.push(res); }`
3. A `generateText` implementation that `shift()`s from the array, returning it, or throwing if empty.
4. A `streamGenerate` implementation that splits the shifted response by spaces and invokes `onToken` synchronously or via a tiny timeout.
5. Export the class.
</action>

- `MockLLMService.ts` contains `public enqueueResponse(res: string)`
</acceptance_criteria>
