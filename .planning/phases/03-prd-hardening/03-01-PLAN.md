## Step 3 — GDPR/HIPAA Auto-Detector

After PRD sections are parsed, run a lightweight keyword scan:

```typescript
function detectComplianceFlags(sections: PRDSection[]): PRDSection[] {
    const gdprKeywords = ['email', 'name', 'address', 'phone', 'user data', 'personal', 'profile', 'location', 'analytics'];
    const hipaaKeywords = ['health', 'medical', 'diagnosis', 'patient', 'prescription', 'clinical', 'symptom'];
    
    return sections.map(s => {
        const flags: string[] = [];
        const lower = s.content.toLowerCase();
        if (gdprKeywords.some(k => lower.includes(k))) flags.push('⚠️ GDPR: Personal data detected — add consent mechanism and data retention policy');
        if (hipaaKeywords.some(k => lower.includes(k))) flags.push('⚠️ HIPAA: Health data detected — encryption at rest and audit logging required');
        return { ...s, complianceFlags: flags };
    });
}
```

Append compliance flags to the Security section content.
